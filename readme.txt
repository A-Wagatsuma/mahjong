これはmjscore形式の牌譜データを入力とし、各局面における各プレイヤの手牌および鳴いている牌を出力するプログラムである。出力はターミナル画面に出力される。同じディレクトリにあるscore.txt, tmp.txtは共にmjscore形式の牌譜である。・実行方法./Mahjong.py [mjscore形式の牌譜データ]・出力画面の読み方(プレイヤを表す数字+行動)プレイヤ1の手牌プレイヤ1の鳴いた牌プレイヤ2の手牌プレイヤ2の鳴いた牌プレイヤ3の手牌プレイヤ3の鳴いた牌プレイヤ4の手牌プレイヤ4の鳴いた牌(プレイヤを表す数字+行動)...・出力画面の略記についてプレイヤを表す数字1:プレイヤ12:プレイヤ23:プレイヤ34:プレイヤ4行動A:上がりG:ツモd:牌を捨てるD:ツモ牌を捨てるC:チーをするN:ポンをするK:カンをする牌m:萬子p:筒子s:索子1z:東2z:南3z:西4z:北5z:白6z:發7z:中5M,5P,5S:赤ドラ
